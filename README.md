# Aivora
Bu proje, kurumsal firmalar için RAG tabanlı yapay zeka destekli müşteri iletişim ve süreç otomasyon platformu geliştirmeyi amaçlamaktadır. Sistem; bilgi tabanı analizi, görev oluşturma, çok kanallı entegrasyon ve otonom aksiyon alma yetenekleri ile klasik chatbot yapılarının ötesine geçerek süreç yöneten bir AI altyapısı sunacaktır.
